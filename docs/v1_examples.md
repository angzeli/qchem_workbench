# v1 Example Compatibility

The v1 example workflows remain valid in v2.0. Existing commands using the
`qchem_workbench` package name and `qchemwb` CLI name continue to work.

The expanded v2 example validation gate covers:

- `examples/basic_molecules/`
- `examples/gaussian_parsing/`
- `examples/orca_parsing/`
- `examples/qe_parsing/`
- `examples/co2rr_molecular/`
- `examples/surface_adsorption/`
- `examples/che_analysis/`
- `examples/screening_campaign/`

All committed parser-output fixtures in these examples are synthetic and are
used for workflow validation only.

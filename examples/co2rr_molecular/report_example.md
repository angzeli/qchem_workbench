# qchem-workbench CO2RR molecular report example

This is a compact report-shape example for the synthetic CO2RR molecular
fixtures. It is not a scientific report.

## Scope

The example demonstrates registry validation, Gaussian input rendering, parsing
synthetic Gaussian-like outputs, reaction-table bookkeeping, and Markdown report
generation.

## Caveat

The workflow keeps electronic energies separate from reaction energies. It does
not apply electrochemical, solvation, standard-state, thermochemical, or
mechanistic corrections.
